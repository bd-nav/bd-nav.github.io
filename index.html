Beyond Destinations: Language-Conditioned Graph Path Planning and Navigation with OpenStreetMap

Anonymous Authors
Code

Abstract

We present BD-Nav, a language-conditioned graph planning and navigation framework that integrates large language models with OpenStreetMap to generate routes aligned with user intent. Unlike prior approaches that treat language primarily as goal specification, BD-Nav interprets free-form instructions to modify both graph feasibility and traversal cost, enabling route-level constraints such as safety preferences, environmental conditions, avoidance rules, and ordered waypoints. The framework combines intent grounding, instruction-aware edge weighting, and semantic path verification within a unified planning pipeline. The resulting instruction-conditioned path is executed on a quadruped robot using SLAM-based localization and control. Extensive experiments across diverse OSM scenarios and real-world outdoor navigation demonstrate improved instruction alignment, robustness to incomplete map data, and reliable execution.
