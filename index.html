Beyond Destinations: Instruction-Aware Graph Path Planning and Navigation with OpenStreetMap

Anonymous Authors
Code

Abstract

We propose BD-Nav, a novel instruction-aware navigation framework that integrates large language models with OpenStreetMap graphs to generate routes that align with user intent. Unlike prior approaches that only extract destinations from language, BD-Nav interprets free-form instructions to identify not only subgoals but also intermediate routes that satisfy conditions such as safety, weather, or avoidance constraints. Our system employs sequential large language model modules for intent parsing, weight function generation, and path evaluation, enabling iterative refinement of graph-based routes. We validate BD-Nav on real-world outdoor experiments with a quadruped robot, demonstrating its ability to follow user instructions without pre-built maps. Results show improved success rates in multi-subgoal scenarios and enhanced robustness in incomplete OpenStreetMap regions compared to ablation variants. This work highlights the potential of combining open-source maps and large language model reasoning to achieve practical, user-centered robot navigation. Code and supplementary materials are available at https://bd-nav.github.io/.
